Color Usage

Background and text colors

When using colors as background and text, be mindful of the background / text contrast. The default color palettes in this style guide primer lend themselves to the following background / text color pairs:

  • Background dark / text white
  • Background base / text white
  • Background light / text dark and
  • Background very light / text base

Background / text color pairs should be evaluated for each project based on its specific colors palettes.

For elements that require background and text color, write mixins as shown below, and use them to extend the elements' classes.
