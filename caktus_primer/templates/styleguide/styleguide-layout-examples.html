Layout Examples

Here are some examples of how you can use our grid system for your various layout needs.

Some best practices to keep in mind:

  • Prefer percentages or viewport units over pixel units.
  • Prefer mobile-first approach when introducing breakpoints.
  • Prefer HTML5 elements over generic div elements.

Hero Full-width, Reverse Column Example

{% example wide=True %} There is no need to move or change elements around in the markup if you need to switch panels around. Simply reverse the direction of your flex container to switch the order of elements. section border: 1px solid #ccc &:nth-child(even) background: white img display: inline-block margin: 0 auto width: 100% .page-container &:nth-child(2n) panels(row-reverse, 0.33, 0.66) article background: #e6eeec aside border-right: 1px solid #ccc background: darkgrey
