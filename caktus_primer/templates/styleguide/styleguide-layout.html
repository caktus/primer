Build grid based layouts with the helpful gridLayout(4) utility.

Simple specify the number of columns in your layout, 4 in this example, and define the spread of the areas. Here we layout a header, nav, main section, and footer on a 4-column grid.

gridLayout() generates both a CSS Grid based version and a Flexbox fallback for you.

.basic-page-grid2 gridLayout(4) >* background: grey border: 1px solid black header gridItem(1, 1, 4, 1) nav gridItem(1, 2, 1, 2) section gridItem(2, 2, 3, 1) footer gridItem(2, 3, 3, 1)

Of course, many layouts need to fit into different spaces. You can combine these tools with simple media queries.

.basic-page-grid3 gridLayout(4) >* background: grey border: 1px solid black header gridItem(1, 1, 2, 1) nav gridItem(3, 1, 2, 1) section gridItem(1, 2, 4, 1) footer gridItem(1, 3, 4, 1) @media (min-width: 800px) header gridItem(1, 1, 4, 1) nav gridItem(1, 2, 1, 2) section gridItem(2, 2, 3, 1) footer gridItem(2, 3, 3, 1)
