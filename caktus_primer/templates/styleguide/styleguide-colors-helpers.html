Color Helpers

Color selection is important and not always obvious, so there are many helpers to make the job easier.

Color Constrast for Web Accessibility

When using colors for text and background, be mindful of the text/background contrast. Aim at achieving contrast ratio of 4.5:1 for normal text and 3:1 for large text (level AA of WCAG 2.0).

A possible exception to meeting the color contrast requirement could be a disabled element, in which dimished legibility of the text is itself an indication of the element's disabled state.

Text/background color pairs should be evaluated for each project based on its specific colors palettes.

Action Colors

Colors can be used to indicate the meaning of interactions. Make sure you donâ€™t use color alone as action indicator, but combine it with other means of conveying information, e.g., text.

The meaning should be assigned to colors based on established conventions and the selection of colors within a given palette. For the color palettes shown in this Styleguid Primer, the suggested meaning of interactions includes:

  • Primary for affirmative and success actions and feedback
  • Secondary for neutral actions
  • Alert for destructive actions, warning and error feedback

Interactive Elements Colors

For interactive elements with larger interactive surface area (e.g. buttons), use the background color that has sufficient contrast with white text. Create hover and focus states by darkening the rest state color, and an active state by increasing the saturation. The exact values of the darken and saturate functions should be project specific.

For disabled elements use mid gray text on very light gray background. This color combination does not meet the minimum color constrast requirement, and creates a friction that provides additional feedback that the element is disabled.

Messaging Colors

For messages, callouts, and feedback, use very light shades for background. Use dark or very dark color for text based on the minimum color contrast requirement (see possible color combinations under Text/Background Color Combinations).

✓ This is a success message!
