Typography

Consistent web typography helps achieve a hierarchy and vertical rhythm of a layout.

Font Size

Base Font Size

The base font size is defined as a variable $font-size-base. The default value for this style guide is $font-size-base = 16px, and it can be adjusted based on project needs.

Responsive Font Sizes

Font sizes across devices scale from a minimum value on mobile, $font-size-very-small, to a maximum value on desktop, $font-size-base, between a minimum screen width, $minScreenWidth = 320, and a maximum screen width, $maxScreenWidth = 960. The minimum to a maximum font size values, as well as the minimum and maximum screen width values can be modified per project requirements.

Responsive font sizes are defined by a graduated() function, which takes a property (e.g. font size), its minimum and maximum values, and a value of the modular scale (see explanation of the modular scale below).

{% example wide=True %} body graduated(font-size, $font-size-small, $font-size-base)

Nullam quis risus eget urna mollis ornare vel eu leo. Etiam porta sem malesuada magna mollis euismod.

{% endexample %}

Vertical Rhythm

Vertical rhythm is a vertical patterns in a layout achieved by a repetition of typographic elements (headings, paragraphs, and lines of text), and the white space between them. We create vertical rhythm by controlling the values of:

  • Modular scale
  • Line-height
  • Margins

Modular Scale

Font size increases from <p> through <h> elements by a constant ratio. The resulting sequence of font size values is called modular scale. The value of the modular scale factor is defined as a variable $fontScale and can be modified per project requirements. The default value used in this style guide primer is major third, $fontScale = 1.25.

Line Height

Line-height is the basis of a so-called baseline grid, a technique fundamental to establishing correct typographic relationships.

In this style guide primer, the base line height is defined as a variable $line-height-base. The default line-height value for this style guide is 1.5 ($line-height-base = $font-size-base*1.5), and it can be adjusted based on project needs.

Responsive Line Height

Line height across devices scales from a minimum value on mobile, $line-height-small, to a maximum value on desktop, $line-height-base, between a minimum screen width, $minScreenWidth = 320, and a maximum screen width, $maxScreenWidth = 960. The minimum to a maximum line height values, as well as the minimum and maximum screen width values can be modified per project requirements.

Responsive line height is defined by a graduated() function, which takes a property (e.g. line height), its minimum and maximum values, and a value of the modular scale (see explanation of the modular scale below).

{% example wide=True %} body graduated(line-height, $line-height-small, $line-height-base)

Margins

Margins applied to typographic elements are derived from the line height. For example, a default margin-bottom on a p element is defined as 1.25 of the base line height ($line-height-base*1.25).

Responsive Margins

Using the graduated() function, we scale margins across devices between a minimum screen width, $minScreenWidth = 320, and a maximum screen width, $maxScreenWidth = 960, from $line-height-small*1.25 to $line-height-base*1.25.

{% example wide=True %} p graduated(margin-bottom, $line-height-small * 1.25, $line-height-base * 1.25)

Nullam quis risus eget urna mollis ornare vel eu leo. Etiam porta sem malesuada magna mollis euismod.

Integer posuere erat a ante venenatis dapibus posuere velit aliquet. Donec sed odio dui. Donec ullamcorper nulla non metus auctor fringilla. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit.

{% endexample %}
