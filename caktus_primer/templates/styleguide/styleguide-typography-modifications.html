Typography Modifications

Modifying font sizes with HTML tag selectors

A semantic <h2> heading in a sidebar may have to be smaller than the default font size of <h2> derived from the modular scale. Text in the footer may have to be smaller than the default <p> font size.

Modify font sizes by writing HTML tag selectors before resorting to writing custom classes.

{% example wide=True %} Modified font size of an h2 heading in a sidebard. aside h2 graduated(font-size, $font-size-small, $font-size-base, 0) font-weight: $font-weight.bold

Lorem Mollis Dapibus

{% endexample %} {% example wide=True %} Modified font size of a paragraph in a footer. footer p font-size: $font-size-small

Cras justo odio, dapibus ac facilisis in, egestas eget quam.

{% endexample %}

Modifying font sizes with custom semantic classes

If writing semantic HTML selectors is not sufficient to overwrite the default font sizes of typographic elements, write custom semantic classes.

{% example wide=True %} A custom semantic class to make the heading in the hero larger than the default

font size. .home-hero-heading graduated(font-size, $font-size-small, $font-size-base, 5) graduated(line-height, $line-height-small, $line-height-small, 5)

Mollis Sem Bibendum

{% endexample %} {% example wide=True %} A custom semantic class to make the tagline in the hero larger than the default

font size. .home-hero-tagline graduated(font-size, $font-size-small, $font-size-base, 1) graduated(line-height, $line-height-base, $line-height-base, 1)

Cras mattis consectetur purus sit amet fermentum.

{% endexample %} {% endblock %}
