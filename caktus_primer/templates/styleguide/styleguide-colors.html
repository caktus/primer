Colors

Color is used to represent the brand and to communicate feedback about user actions.

Brand colors are expressed as primary, secondary, and accent colors. For brands with a more extensive color palette, additional designations are needed.

Action colors may be derived from brand colors if appropriate. They communicate main categories of feedback about user actions: success (as well as other types of affirmative actions, e.g. confirmation), alert (as well as destructive actions, e.g. delete), and neutral, e.g. informative feedback with neutral effect.

Grayscale

Grayscale has a number of applications. Most common include communicate a disabled state of an element and creating a table zebra pattern.

Grayscale consists of black and white, and five shades that are alpha values between them, expressed as Stylus hash objects. Grayscale can be extended to include more shades as necessary.

Black, white, and the the shade of gray are retrieved from hash objects by concatenating the hash name, key, and nested key, e.g. $palette.grayscale.dark.

Grayscale Range

  • $palette.grayscale.black
  • $palette.grayscale.very-dark
  • $palette.grayscale.dark
  • $palette.grayscale.mid
  • $palette.grayscale.light
  • $palette.grayscale.very-light
  • $palette.grayscale.white

Disabled

  • $palette.disabled.light
  • $palette.disabled.very-light
